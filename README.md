# Ordinex

Ordinex is a desktop app that turns messy folders into organized spaces in minutes.

## What Ordinex Does

- Scans folders and understands what each file is.
- Suggests where files should go before any move happens.
- Lets you review everything first, then organize in one action.
- Keeps a history of actions so you can roll back if needed.

## Why People Use It

- Save time spent manually sorting downloads, screenshots, documents, and media.
- Keep projects clean without creating fragile manual routines.
- Reduce the risk of accidental file moves with preview and rollback.

## Core Experience

1. Choose a folder.
2. Review suggested categories and destinations.
3. Apply sorting with confidence.
4. Undo if anything needs to be restored.

## Built For Real-World Workflows

- Works well for creators, students, professionals, and anyone managing large folders.
- Handles recurring clutter and one-time cleanups.
- Keeps your preference choices so the app becomes smarter for your setup.

## Updates

- Ordinex provides regular releases with installer updates.
- Release notes are automatically generated from commit history so each release clearly shows what changed.
